Welcome to fleq!

  • IMPORTANT
  • FLEQ games are a real time application, so connection is very important to get a good game experience. Please, ensure that you have a good wireless connection before to start a game in FLEQ (use WiFi if you can). The application uses last web technologies, like WebSocket provided by HTML5. This feature only works on recent browser versions (like Mozilla Firefox on Android). You can check if your browser support Websocket in http://www.websocket.org/echo. If your browser can't use this technology, it will use another less quick like AJAX Long Polling which introduces a little delay in comparision with WebSockets. Therefore, use a browser that support WebSocket (if available) and, the most important, use WiFi connection always that you can do it.
